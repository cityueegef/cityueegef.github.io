Project ID : P30
Defect Detection in Irwin mangoes (Mangifera indica)
Project InformationPoster
P30
Defect Detection in Irwin mangoes (Mangifera indica)
Student Names:
Zachary Tang
Video Demo Not Available
GitHub
Farmers, wholesalers and retailers around the world often rely on human labor to verify a fruit's saleability, leading to high percentages of costly food waste, supply chain carbon emission and unnecessary packaging. This project aims to develop a defect detection model in YOLOv8 to identify predetermined defects such as discolouration and bruising, initially using Irwin mangoes (Mangifera indica) common in Taiwan, to help address this issue by identifying defects faster, cheaper, and more accurately.
